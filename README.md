# 模擬試験クイズアプリ

模擬試験191問をブラウザ上で解答できるクイズアプリケーションです。

## ファイル構成

- `index.html` - メインのHTMLファイル
- `styles.css` - スタイルシート
- `app.js` - アプリケーションのロジック
- `questions.json` - 問題データ（JSON形式）

## 使い方

1. `index.html` をブラウザで開きます
2. 問題を選択して解答します
3. 前の問題/次の問題ボタンで移動できます
4. 問題一覧から直接問題にジャンプできます

## 機能

- 191問すべての問題を表示
- 進捗バーで進捗を確認
- 問題一覧から直接ジャンプ
- 選択肢の選択と保存
- **正解表示機能**: 「正解を表示」ボタンで正解と解説を表示
- **正解/不正解の視覚的表示**: 正解表示モードでは、正解は緑、不正解は赤で表示
- 問題一覧で正解/不正解の状態を確認可能
- キーボードショートカット対応
  - 左矢印キー: 前の問題
  - 右矢印キー: 次の問題
  - 1-4キー: 選択肢を選択
  - Aキー: 正解表示の切り替え

## 正解表示機能

「正解を表示」ボタンをクリックすると、現在の問題の正解と解説が表示されます。正解表示モードでは：
- 正解の選択肢が緑色でハイライト表示されます
- 選択した不正解の選択肢が赤色で表示されます
- 問題一覧でも正解/不正解の状態が色分けで表示されます

## 問題データの追加

`questions.json` に問題を追加することで、新しい問題を追加できます。

問題の形式:
```json
{
  "id": 問題番号,
  "question": "問題文",
  "options": ["選択肢1", "選択肢2", "選択肢3", "選択肢4"],
  "correctAnswer": 正解のインデックス（0-3）
}
```
